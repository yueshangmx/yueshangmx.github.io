### 1.与变形/转换相关的CSS属性
```
    transform:  none / 2d-3d-func;
    transform:  func1()  func2()  func3();
    transform-origin: left/center/right  top/center/bottom;  #指定变形的原点
```
`注意`<br>
`1. 所有的变形效果都不会释放或改变原始占用的空间！——与相对定位有点类似。`<br>
`2. 注意Safari的兼容性问题。`
